# Fortgeschrittene-Programmierung-Uebungen
Zum nutzen bitte folgende Steps befolgen:
1. Repository von GitHub local auf eigenen PC klonen
2. Den Ordner mit PyCharm oder VSCode als neues Projekt öffnen
3. Ein virtuelles Environment erstellen (ich habe python 1.14 benutzt aber alles über 1.11 sollte locker klar gehen)
4. Folgenden Befehl in die Konsole des Virtuellen Environments eingeben: pip install -r requirements.txt
5. Installieren lassen und Fertig
